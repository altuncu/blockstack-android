validateProofs

fun validateProofs(profile: Profile, ownerAdress: String, name: String? = null, callback: (Result<ArrayList<Proof>>) -> Unit): Unit

Validates the social proofs in a user's profile. Currently supports validation of Facebook, Twitter, GitHub, Instagram, LinkedIn and HackerNews accounts.

Parameters

profile - The profile to be validated

ownerAddress - The owner bitcoin address to be validated

name - (default = null) The Blockstack name to be validated

callback - called with a list of validated proof objects or an error
