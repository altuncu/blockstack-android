Executor

interface Executor

Executor defines where functions are executed. Three different cases are distinguished:

  1. main thread: to start an intent launching the login process. This has to be the UI thread.
  2. network thread: to make network calls. This must not be the UI thread, it is usually a thread from CommonPool.
  3. v8 thread: to continue after network calls. This must be on the thread that is currently used by the j2v8 engine.
